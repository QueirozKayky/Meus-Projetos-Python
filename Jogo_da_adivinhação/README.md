# Jogo da Adivinhação em Python

Um jogo simples onde o usuário tenta adivinhar um número secreto, recebendo dicas de "muito alto" ou "muito baixo". Este projeto foi fundamental para o aprendizado de loops (`while`), lógica condicional e a interação contínua com o usuário.

## Funcionalidades

- Gera um número secreto aleatório entre 1 e 100.
- Dá dicas ao jogador a cada tentativa.
- Conta o número de tentativas até o acerto.
- Encerra o jogo quando o número é adivinhado.

## Como Executar

1.  Navegue até a pasta `jogo_da_adivinhacao` no seu terminal.
2.  Execute o script com o comando:
    ```bash
    python jogo_da_adivinhacao.py
    ```

## Conceitos Praticados

- `random`: Para escolher o número secreto.
- `while`: Para criar o loop do jogo.
- `if/elif/else`: Para a lógica das dicas.
- `input()`: Para a entrada do palpite do usuário.
