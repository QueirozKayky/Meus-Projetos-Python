# Gerenciador de Tarefas em Python

Este é um gerenciador de tarefas simples, feito em Python, que roda no terminal. O projeto foi o meu primeiro passo no aprendizado da linguagem, focado em entender os conceitos básicos de listas, loops e menus interativos.

## Funcionalidades

- Adicionar novas tarefas à lista.
- Visualizar todas as tarefas salvas.
- Deletar tarefas existentes.
- Sair do sistema.

## Como Executar

1.  Navegue até a pasta `lista_de_tarefas` no seu terminal.
2.  Execute o script com o comando:
    ```bash
    python lista_de_tarefas.py
    ```

## Conceitos Praticados

- `Listas`: Para armazenar as tarefas.
- `Loops`: Para manter o programa em execução.
- `Menus Interativos`: Para guiar a interação com o usuário.
- `Entrada de Usuário`: Para receber os comandos e tarefas.
