# Gerador de Senhas em Python

Este é um gerador de senhas personalizável feito em Python. O projeto foi desenvolvido para praticar os conceitos de funções, lógica condicional e a manipulação de strings.

## Funcionalidades

- Gera senhas personalizadas com base nas escolhas do usuário.
- Permite escolher o tamanho da senha.
- Inclui opções para letras minúsculas, maiúsculas, números e símbolos.
- Trata erros de entrada do usuário, garantindo um uso seguro.

## Como Executar

1.  Navegue até a pasta `gerador_de_senhas` no seu terminal.
2.  Execute o script com o comando:
    ```bash
    python gerador_de_senhas.py
    ```

## Conceitos Praticados

- `random`: Para a geração de caracteres aleatórios.
- `input()`: Para interagir com o usuário.
- `Funções`: Para organizar o código em blocos lógicos.
- `Manipulação de Strings`: Para construir a senha final.
