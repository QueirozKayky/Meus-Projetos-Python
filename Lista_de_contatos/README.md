# Sistema de Gerenciamento de Contatos em Python

Este é um sistema de gerenciamento de contatos baseado em console. O projeto foi desenvolvido para praticar o uso de **listas** e **dicionários**, duas das estruturas de dados mais importantes em Python.

## Funcionalidades

- Adicionar novos contatos com nome, telefone e e-mail.
- Listar todos os contatos salvos.
- Buscar por um contato específico usando o nome.
- Interface intuitiva e simples de usar, baseada em um menu.

## Como Executar

1.  Navegue até a pasta `lista_de_contatos` no seu terminal.
2.  Execute o script com o comando:
    ```bash
    python lista_de_contatos.py
    ```

## Conceitos Praticados

- `Dicionários`: Para armazenar as informações de cada contato.
- `Listas de Dicionários`: Para gerenciar múltiplos contatos.
- `Lógica de Busca`: Para percorrer a lista e encontrar o contato desejado.
- `Estruturas de Dados`: Demonstração prática do uso de estruturas de dados complexas.
