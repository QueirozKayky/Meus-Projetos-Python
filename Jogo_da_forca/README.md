# Jogo da Forca em Python

Uma versão do famoso jogo da forca para o terminal. Este projeto foi um desafio maior, combinando loops, listas, manipulação de strings e uma lógica mais complexa para verificar a vitória e a derrota.

## Funcionalidades

- Seleciona uma palavra secreta aleatória de uma lista pré-definida.
- Exibe o progresso do jogador com hífens ou asteriscos.
- Controla o número de tentativas restantes.
- Informa ao jogador se a letra está na palavra ou não.
- Revela a palavra no final do jogo, em caso de derrota.

## Como Executar

1.  Navegue até a pasta `jogo_da_forca` no seu terminal.
2.  Execute o script com o comando:
    ```bash
    python jogo_da_forca.py
    ```

## Conceitos Praticados

- `Listas`: Para a lista de palavras e para o progresso do jogador.
- `Loops aninhados`: Para iterar sobre as letras da palavra e verificar o palpite.
- `Tratamento de Erros`: Para validar a entrada do usuário.
- `Controle de Fluxo`: Para gerenciar as condições de vitória e derrota.
